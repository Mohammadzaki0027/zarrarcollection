# 🌸 ROSA Mode — Women's Dress Collection Website

A beautiful, fully-featured women's fashion website built with **React + Tailwind CSS**.

## ✨ Features

- **4 Pages**: Home, Products, Gallery, Contact
- **Hero Carousel** — auto-advances every 4.5s with animated catalogue preview
- **Dark / Light Theme** toggle with smooth transitions
- **Catalogue Search** — filter by name, style, colour
- **Category Filters** — Gown, Midi, Mini, Maxi, Cocktail, Casual
- **Price Range Slider** — live filtering
- **Sort** — by featured, price, or rating
- **Pagination** — 6 products per page
- **1-column layout on mobile**, 2-col on tablet, 3-col on desktop
- **Gallery Lightbox** — click to expand
- **Wishlist hearts** per card (no cart)
- **Rich animations** — fade, slide, zoom, float, shimmer, heartbeat
- **Mobile-responsive** navbar with hamburger menu
- **Marquee ticker** strip

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ installed

### Install & Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm run preview
```
